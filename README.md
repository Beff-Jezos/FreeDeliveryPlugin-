# FreeDeliveryPlugin-
(WIP)
Shopware6 Plugin that displays the remaining amount for free shipping. <strong>Highly customizable in the Plugin Config for the User.</strong> <br>
Choose between different Styles to display the progress like "ProgressBar + Icon" or "nothing" if you prefere to only activate specific spots!
Available on the product page, Offcanvas Cart, Cart & Checkout Page.
